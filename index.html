About me

I am a postdoc at UC Berkeley in the Gallant Lab. I defended my Ph.D. in 2018, in the Image, Data, Signal department at Telecom ParisTech in France, supervised by Alexandre Gramfort and Yves Grenier. I graduated from Ecole polytechnique in 2013 and EPFL in 2015.

My work focuses on signal processing and machine learning for neuroscience.

I am also a core developer of scikit-learn since 2015.

Software

scikit-learn

Machine Learning in Python - Core developer
Implementation of optimization algorithms (SAG, NMF), API design for nearest-neighbor-based estimators, library maintenance.

pactools

Phase amplitude coupling (PAC) in Python - Main developer
Estimation of PAC in neural time series. Implementation of driven auto-regressive (DAR) models.

alphacsc

Convolutional sparse coding in Python - Core developer
Different models of convolutional sparse coding, including univariate and multivariate models, Gaussian and alpha-stable noise distributions.

himalaya

Multiple-target linear models - Main developer
Regularized linear models in Python, focusing on computational efficiency for large numbers of targets.

voxelwise_tutorials

Tutorial for the voxelwise modeling framework - Main developer
Voxelwise modeling is a framework to perform functional magnetic resonance imaging (fMRI) data analysis, fitting encoding models at the voxel level.
